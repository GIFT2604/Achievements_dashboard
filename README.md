# Achievements_dashboard
User-Chievements-Dashboard
React + Tailwind Achievements UI for the SumiCode platform. Staff users can view the users', courses, enrollments, progress, certificates, exercises, and tasks.

Setup


Install and run:

npm install
npm run dev
Open http://localhost:5175/ 


Scripts
npm run dev – development server
npm run build – production build
npm run preview – preview production build
